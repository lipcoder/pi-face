## 人脸识别系统

------

人脸识别核心来自于https://github.com/HyperInspire/InspireFace

单项目内当前使用https://github.com/lipcoder/InspireFace，
对原项目进行了删减和改动，构建由docker换为podman，计划更新为pod启动

当前项目正在开发中，如有需要，请先使用版本v1.0.0
